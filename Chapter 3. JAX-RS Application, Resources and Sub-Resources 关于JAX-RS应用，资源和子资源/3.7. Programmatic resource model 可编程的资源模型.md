3.7. Programmatic resource model 可编程的资源模型
========================

资源可以由类或实例也可以由可编程的资源模型构造。每一个资源从资源类创建也可以使用编程API构建资源生成器。更多信息请参考 [资源生成器](https://jersey.java.net/documentation/latest/resource-builder.html) 部分。
