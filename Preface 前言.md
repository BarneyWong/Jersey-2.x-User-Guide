Preface 前言
========================

这是Jersey 2.10.1 的用户指南。我们极力将它能与我们新增的功能保持一致。当阅读本指南，作为补充，也请移步至[ Jersey API documentation ](https://jersey.java.net/apidocs/2.10.1/jersey/index.html)查看 Jersey 的特性和 API。
